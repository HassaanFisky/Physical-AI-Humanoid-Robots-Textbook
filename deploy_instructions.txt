## Troubleshooting

**Still 404?**
- Clear Vercel cache: Settings ‚Üí Advanced ‚Üí Clear Cache
- Check build logs for errors
- Verify `book-site/build/index.html` exists after build

**Build fails?**
- Check Node.js version >= 18
- Verify `package.json` has all dependencies
- Review build logs in Vercel dashboard
