# Prompt: Generate Chapter Content

**Goal**: Write a new chapter for the Physical AI Textbook.

**Context**:

- Audience: Engineering students / Enthusiasts
- Tone: "Smart Student", enthusiastic but rigorous.
- Format: Docusaurus Markdown (MDX) with Admonitions.

**Instructions**:

1. Topic: [INSERT TOPIC] (e.g., "Inverse Kinematics for Humanoids")
2. Structure:
   - **Introduction**: Hook the reader. Why does this matter?
   - **Theory**: The math/physics (use LaTeX `$$`).
   - **Implementation**: Code snippets (Python/C++ for ROS 2).
   - **Real World**: Example (Tesla Optimus, Atlas).
3. Requirements:
   - Use `:::tip`, `:::warning` for callouts.
   - Include Mermaid diagrams for logic flows.
   - Keep paragraphs short and punchy.

**Output format**: Complete `.md` file content.
