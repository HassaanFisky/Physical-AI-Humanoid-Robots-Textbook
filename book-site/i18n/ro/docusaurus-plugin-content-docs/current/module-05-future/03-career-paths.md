# Career Paths

![Robotics Engineer](https://images.unsplash.com/photo-1485827404703-89b55fcc595e?w=1200&h=600&fit=crop)

> **Point ki Baat:** Humanoid robotics tech dunya ke sab se tez barhte huay shobon mein se aik hai. Career landscape ko samajhna aapki planning mein madad kare ga.

## Introduction: Robotics ka Sunehri Daur (Golden Age)

Robotics industry mein is waqt kabil engineers ki boht demand hai. Companies achi salaries aur baray projects offer kar rahi hain.

---

## Core Roles (Aham Kirdar)

### Software Engineer (Robotics)
**Zimmedari**: Control algorithms banana aur systems ko jorna.
**Skills**: C++, Python, ROS 2.

### Perception Engineer
**Zimmedari**: Computer vision aur sensor fusion par kaam karna.
**Skills**: Deep Learning, OpenCV.

### Controls Engineer
**Zimmedari**: Balance aur locomotion ke systems banana.
**Skills**: Control Theory, Optimization.

---

## Skills Roadmap

1. **Base**: Programming (Python aur C++) seekhein.
2. **ROS 2**: Robot Operating System par command hasil karein.
3. **Projects**: Simulation mein apne projects banayein.
4. **Specialization**: Vision, Control ya AI mein se aik field choose karein.

---

## Educational Resources

- **Courses**: Coursera aur MIT ke online courses.
- **Books**: "Modern Robotics" aur "Probabilistic Robotics".
- **Practice**: ROS 2 tutorials aur Isaac Sim.

---

## Aham Nukat

:::note Khulasa (Summary)

1. **Demand boht zyada hai**: Sahi skills wale logon ki kami hai.
2. **Main Fields**: Software, Vision, aur Control.
3. **Projects**: Skill prove karne ke liye projects zaroori hain.
4. **Stay Updated**: Tech boht fast badal rahi hai.
   :::
